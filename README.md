# Outil OSINTT

Outil d'OSINT développé en Python pour la recherche d'informations.

## Fonctionnalités
- Recherche par numéro de téléphone
- Recherche par email
- Recherche par username
- Géolocalisation IP
- Recherche de documents publics

## Installation
```bash
git clone https://github.com/AzouC/Outils-Osintt.git
cd Outils-Osint
pip install -r requirements.txt
